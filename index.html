Fluxgrid

Responsive and customisable utility-based CSS grid built with Sass.

See on GitHub

Columns: 12
Gutters width: 1rem
Nestable: Yes
Direction: Row
Breakpoints:

Extra small Small Medium Large Extra large
Device Phone Tablet Desktop Large desktop Extra large desktop
Breakpoint > 0 ≥ 425px ≥ 768px ≥ 1024px ≥ 1366px
Suffix @xs @sm @md @lg @xl

Syntax

<div class="container container--modifier-*">
	<div class="grid grid--modifier-*">
		<div class="cell cell--modifier-*"></div>
	</div>
</div>
Responsive: *@xs|@sm|@md|@lg|@xl

#Container

A .container is required to cancel the .grid negative margins used to align the .cells. It can be extended with modifiers throughout breakpoints:

.container--* &.container--noGutter
